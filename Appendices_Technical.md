## Apêndice A: Nomogramas Resumidos

### A.1. Nomograma PresbyCor/Custom-Q (Ghenassia)

**Q-Target em Função de Add Desejada**

| Add Desejada | Q-Target Olho Dominante | Q-Target Olho Não-Dominante |
|--------------|------------------------|----------------------------|
| +1.25 D | -0.50 | -0.60 |
| +1.50 D | -0.60 | -0.70 |
| +1.75 D | -0.70 | -0.80 |
| +2.00 D | -0.80 | -0.90 |

**Compensação Esférica:**

$$S_{comp} = |\Delta Q| \times K_{médio} \times 0.75$$

**Exemplo:**
- K médio = 43.00 D
- Q pré-op = -0.25
- Q target = -0.70
- ΔQ = -0.45
- S_comp = 0.45 × 43 × 0.75 = **+14.5 D** (induzir hipermetropia para compensar)

## Apêndice C: Checklists Cirúrgicas (1-Página)

### C.1. Checklist Pré-Op Universal

**Consulta Inicial:**
- [ ] Refração cicloplegiada (se <50 anos ou hipermétrope)
- [ ] Dominância ocular (hole-in-card + convergence)
- [ ] Add necessária (leitura trial com +1.50, +2.00, +2.50 D)
- [ ] Topografia corneana (Pentacam/Galilei)
- [ ] BAD-D ≤1.6 (screening ectasia)
- [ ] Paquimetria central >480 μm
- [ ] RSB estimado >320 μm
- [ ] Aberrometria (coma <0.30 μm, SA baseline)
- [ ] Avaliação cristalino: LOCS, OSI, densitometria
- [ ] OSDI (olho seco): <40
- [ ] TBUT >7 segundos
- [ ] Consentimento informado presbiópico assinado

---

### C.2. Checklist Intra-Op LASIK Presbiópico

**Preparação:**
- [ ] Paciente confirmado (nome, olho a operar)
- [ ] Cálculos verificados (Q-target, sphere, add)
- [ ] Laser programado e verificado
- [ ] Eye-tracker calibrado e testado

**Flap:**
- [ ] Marcação limbal (orientação)
- [ ] Criação flap (espessura target: 100-110 μm)
- [ ] Levantamento smooth
- [ ] Hidrodesidratação estromal (30-60 seg)

**Ablação:**
- [ ] Centragem verificada (Purkinje/Pupil conforme protocolo)
- [ ] Fixação paciente estável >5 segundos
- [ ] Tracking verde/ativo durante ablação
- [ ] Se tracking falha: **PARAR**

**Finalização:**
- [ ] Irrigação interface (BSS)
- [ ] Reposição flap com stretching
- [ ] Secagem margens (Merocel)
- [ ] Striae check (nenhuma visível)
- [ ] Antibiótico + NSAID tópico

---

### C.3. Checklist Follow-Up Pós-Op

**Dia 1:**
- [ ] UCDVA/UCNVA (baseline inicial)
- [ ] Refração (se UCVA <20/60, verificar)
- [ ] Flap check: Posição, sem striae, interface clara
- [ ] Prescrição: Antibiótico 4×/dia, NSAID 4×/dia, Lágrimas 6×/dia

**Semana 1:**
- [ ] UCDVA/UCNVA (melhoria esperada)
- [ ] Refração (estabilização inicial)
- [ ] Interface: DLK? (se sim, ver Capítulo 11)
- [ ] Gestão expectativas: "Visão estranha = normal, vai melhorar"

**Mês 1:**
- [ ] UCDVA/UCNVA (refraining from refraction se possível)
- [ ] Topografia (verify centragem)
- [ ] Decisão: Aguardar neuroadaptação vs. enhancement
- [ ] **Regra:** NÃO enhancement antes mês 6 exceto erro grosseiro

**Mês 3:**
- [ ] UCDVA/UCNVA (deve estar ≥80% final)
- [ ] Refração estável
- [ ] Satisfação score /10
- [ ] Se <6/10: Investigar causa (óptica vs neuroadaptativa)

**Mês 6-12:**
- [ ] Outcome final
- [ ] Alta ou enhancement se necessário
