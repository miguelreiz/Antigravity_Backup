# Guia: Converter para E-Book Digital (ePub/PDF)

## 🎯 OBJETIVO
Criar versão e‑book do livro PresbyCor em formato ePub (Kindle, Apple Books) ou PDF profissional navegável.

---

## ✅ OPÇÃO 1: Converter Word → PDF (MAIS FÁCIL) ⭐

**Você já tem:** `PresbyCor_COMPLETE_BOOK.docx`

### Passo-a-Passo:

1. **Abrir Word:**
   - Abre `PresbyCor_COMPLETE_BOOK.docx` no Microsoft Word

2. **Exportar como PDF:**
   - Menu: `Arquivo` → `Exportar` → `PDF`
   - OU: `Arquivo` → `Salvar Como` → Formato: `PDF`
   
3. **Configurações importantes:**
   - ✅ Marcar: "Incluir marcadores" (cria índice clicável)
   - ✅ Marcar: "Otimizar para: Distribuição online"
   - Nome: `PresbyCor_Complete_Book.pdf`
   - Salvar em: Desktop

4. **Resultado:**
   - PDF navegável com índice
   - Tamanho: ~30-50 MB
   - Compatível: Mac, Windows, iPad, Android

**Tempo:** 2 minutos

---

## ✅ OPÇÃO 2: Converter Word → ePub (para Kindle/iBooks)

### Método A - Usando Calibre (Grátis)

1. **Instalar Calibre:**
   - Download: https://calibre-ebook.com/download
   - Instalar normalmente

2. **Converter:**
   - Abrir Calibre
   - `Adicionar livros` → Selecionar `PresbyCor_COMPLETE_BOOK.docx`
   - Clicar direito no livro → `Converter livros`
   - Formato saída: `EPUB`
   - Preencher metadados:
     - Título: PresbyCor: Modern Strategies for Presbyopia
     - Autor: Dr. Miguel Reis
     - Idioma: Portuguese
   - `OK` para converter

3. **Resultado:**
   - Arquivo ePub pronto
   - Compatível: Kindle, Apple Books, Kobo, etc.

**Tempo:** 5 minutos (primeira vez)

---

### Método B - Usando Pages (Mac nativo)

1. **Abrir no Pages:**
   - Clique direito em `PresbyCor_COMPLETE_BOOK.docx`
   - "Abrir com" → Pages

2. **Exportar:**
   - Menu: `Arquivo` → `Exportar para` → `ePub`
   - Preencher informações:
     - Título, Autor, Idioma
   - `Próximo` → Salvar

3. **Resultado:**
   - ePub Apple (pode não funcionar perfeitamente em Kindle)

**Tempo:** 3 minutos

---

## ✅ OPÇÃO 3: PDF com Índice Navegável (PREMIUM)

### Usando Preview/Adobe Acrobat:

Se você quiser um PDF MUITO PROFISSIONAL com:
- Índice lateral clicável
- Marcadores
- Links internos funcionando

**Passos:**
1. Converter Word → PDF (Opção 1)
2. Abrir PDF no Adobe Acrobat Pro (se tiver)
3. Ferramentas → Editar PDF → Adicionar marcadores
4. Organizar índice navegável

**Ou simplesmente:** O PDF gerado do Word já tem índice funcionando!
