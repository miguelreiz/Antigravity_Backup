Segue um upgrade pragmático do nomograma PresbyCor, incorporando as variáveis técnicas que a investigação revelou (idade, ceratometria K, diâmetro de zona, pupila, anisometropia‑limite e Q pré‑op). O objetivo é manter o algoritmo simples de aplicar em planilha ou App Script, mas mais fiel ao comportamento real do PresbyCor.

1 ▪ Nova lógica de cálculo (presbycor_v2)

python Copiar def presbycor_v2(esf, q0, add, dom, strat, idade, k_med, zona_mm, pupila_mm, aniso_max=1.75, teto_esf=5.0): """ Cálculo refinado do nomograma PresbyCor. Parâmetros novos: idade, K médio, zona óptica, pupila, anisometropia máxima. Retorna (esfera_final, q_target) """ # 1. Normalização de flags dom = str(dom).lower() in ['sim', 'yes', 'true', '1'] strat = str(strat).lower() # 2. Delta‑base para cada estratégia (tabelado) if strat == 'equi-vision': delta_base = 1.2 if dom else 1.8 # mesma lógica anterior q_shift = 0.8 # Q mais negativo bilateral elif strat == 'dual-vision': delta_base = 1.2 if dom else 2.2 q_shift = 0.8 elif strat == 'mono-vision': delta_base = 0 if dom else float(add) # defocus no não‑dominante q_shift = 0 if dom else 0.7 else: # fallback delta_base, q_shift = 0, 0 # 3. Ajustes finos conforme nova pesquisa $# 3a. Idade (≤50 anos = mais acomodação, menor delta)$ delta_idade = -0.1 if idade <= 50 else 0 q_idade = -0.05 if idade <= 50 else 0 # -0,05 = 0,05 mais negativo? $# 3b. Ceratometria (K >45 D → tendência a hipercorreção: reduzir$ delta) delta_k = -0.05 if k_med > 45 else 0.05 if k_med < 42 else 0 $# 3c. Zona óptica (≥7 mm exige maior compensação)$ delta_zona = 0.10 if zona_mm >= 7 else -0.05 if zona_mm <= 6 else 0 $# 3d. Pupila mesópica (>6 mm → reduzir Q para evitar halos)$ q_pupila = -0.15 if pupila_mm > 6 else 0 # 4. Soma dos ajustes delta_total = delta_base + delta_idade + delta_k + delta_zona q_total = q_shift + q_idade + q_pupila # 5. Limitar anisometropia (só se estratégia gera defocus entre olhos) if strat in ['dual-vision', 'mono-vision'] and not dom: # diferença entre novo alvo e olho dominante presumido 0 D if abs(delta_total - (1.2 if strat=='dual-vision' else add)) > aniso_max: # aparar excesso excesso = abs(delta_total) - aniso_max delta_total -= excesso * (1 if delta_total > 0 else -1) # 6. Cálculo final esf_final = min(esf + delta_total, teto_esf) q_target = q0 - q_total return round(esf_final, 2), round(q_target, 2)

Principais ajustes incorporados

Variável Efeito resumido Motivo clínico $Idade ≤50 a → −0,10 D de delta e −0,05 de ΔQ$ Maior acomodação residual, precisa de menos “potência extra”. $K médio >45 D → −0,05 D; <42 D →$ +0,05 D Córneas curvas supercorrigem hipermétropes; planas subcorrigem. $Zona óptica ≥7 mm → +0,10 D; ≤6 mm →$ −0,05 D Zonas grandes exigem mais ablação; pequenas exigem menos. $Pupila mesópica >6 mm → ΔQ −0,15 (menos$ negativo) Reduz halos e perda de contraste em pupilas grandes. Anisometropia‑limi te Corta defocus extra se exceder aniso_max (padrão 1,75 D) Mantém conforto binocular. Ajuste os coeficientes conforme ganhar experiência; eles partem da literatura/empiria de Ghenassia.

Benefícios do upgrade
  • Melhora a exatidão (compensa idade, K e zona de tratamento).
  • Reduz chances de ajuste pós‑op (menor under/over‑correction).
  • Inclui segurança binocular (anisometropia‑cap) e conforto noturno (pupila).
  • Mantém simplicidade – apenas alguns coeficientes fáceis de calibrar.
