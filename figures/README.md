## 🎨 Como Usar

### Visualizar no GitHub
1. Navegue até qualquer arquivo `.md`
2. **Mermaid renderiza automaticamente** no GitHub
3. Flowcharts aparecem como diagramas visuais

### Editar
1. Acesse https://mermaid.live
2. Copie/cole o código do bloco ```mermaid```
3. Edite visualmente
4. Copie código atualizado de volta

### Exportar para PNG/SVG
1. Abra em https://mermaid.live
2. Menu "Actions" → "PNG" ou "SVG"
3. Download alta resolução
