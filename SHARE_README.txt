CONTEÚDO DESTE PACOTE
====================================================================

Este pacote contém o livro técnico completo "PresbyCor" em formato
digital, incluindo 13 capítulos técnicos, prefácio, bibliografia,
glossário, apêndices e 26 infográficos.

ESTRUTURA:
----------

📄 DOCUMENTOS PRINCIPAIS:
- Preface_Methodology.md          (Prefácio + Metodologia Editorial)
- Chapter_1_Complete.md            (Fundamentos Presbiopia - INGLÊS)
- Chapter_2_Complete.md até 13    (Capítulos técnicos - PORTUGUÊS)
- Bibliography_Consolidated.md    (37 referências consolidadas)
- Glossary_Abbreviations.md       (45 termos + 32 abreviações)
- Appendices_Technical.md         (Nomogramas, checklists, templates)
- About_Author.md                 (Sobre o autor)
- README_Book_Structure.md        (Estrutura completa do livro)

📊 ARQUIVO WORD:
- PresbyCor_COMPLETE_BOOK.docx    (Todo livro num único .docx)
- PresbyCor_FULL_BOOK.md          (Todo livro num único .md)

🎨 INFOGRÁFICOS (pasta /figures):
- 15 Flowcharts Mermaid (.md)     (Renderizam no GitHub)
- 5 Gráficos Científicos (.png)   (Python/Matplotlib, 300 DPI)
- 3 Diagramas Técnicos (.png)     (Perfis Q, Ray tracing, LASIK)
- 2 Imagens Conceituais (.png)    (PRESBYOND, RSB)
- Scripts Python (.py)             (Reproduzíveis)
- BioRender_Guide.md              (Guia para criar diagramas futuros)

COMO USAR
====================================================================

OPÇÃO A - LEITURA RÁPIDA:
1. Abrir "PresbyCor_COMPLETE_BOOK.docx" no Microsoft Word
2. Ver livro completo formatado

OPÇÃO B - NAVEGAR POR CAPÍTULO:
1. Abrir cada "Chapter_X_Complete.md" em editor Markdown
   (Recomendo: Typora, Visual Studio Code, ou simplesmente
    abrir no GitHub)
2. Flowcharts Mermaid renderizam automaticamente no GitHub

OPÇÃO C - VER NO GITHUB:
1. Acessar: https://github.com/miguelreiz/PresbyCor-Book
2. Todos os flowcharts renderizam visualmente
3. Navegação web fácil
